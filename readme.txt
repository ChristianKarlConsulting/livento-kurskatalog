=== Livento Kurskatalog ===
Contributors: livento
Requires at least: 6.0
Tested up to: 6.7
Requires PHP: 7.4
Stable tag: 1.7.5
License: Proprietär

Rendert den oeffentlichen Kurskatalog aus Campus Connect nativ in WordPress.

Shortcodes:
* [livento_kurse] - voller Katalog mit Filtern
* [livento_kurse limit="6" sort="popular"] - kuratierter Block ohne Filter
* [livento_kurse_suche] - eigenstaendiges Suchfeld
